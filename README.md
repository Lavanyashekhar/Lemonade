🍋 Lemonade App – Jetpack Compose Project
📱 Overview

The Lemonade App is a simple interactive Android application built using Jetpack Compose. The app simulates the process of making lemonade in a fun and engaging way by guiding users through multiple steps with images and click interactions.

This project demonstrates state management, composable functions, and Material 3 theming in Android.

🚀 Features

🍋 Step-by-step lemonade making process

🖼️ Image-based UI using drawable resources

🔁 Interactive click-based state changes

🎨 Material 3 theming support (Light/Dark mode)

🧩 Built entirely using Jetpack Compose

🛠️ Tech Stack

Kotlin

Jetpack Compose

Material 3

Android Studio

📂 App Flow

Lemon Tree – Tap the tree to pick a lemon

Squeeze Lemon – Tap multiple times to squeeze

Drink Lemonade – Tap to drink

Restart – Tap to start again
